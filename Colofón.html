Colofón

La producción de este libro se realizó utilizando el modelo de edición ramificada con herramientas de software libre, el trabajo de edición y composición tipográfica se realizó con el lenguaje LaTeX.

Sobre la versión PDF para imprenta

Las familias tipográficas utilizadas dentro del libro son: IBM Plex, una superfamilia de tipografía abierta, diseñada y desarrollada conceptualmente por Mike Abbink en IBM con colaboración de Bold Monday y Minion un tipo serif, lanzada en 1990 por Adobe Systems. Diseñado por Robert Slimbach, está inspirada en el tipo de época del tardío Renacimiento y destinado al texto del cuerpo y la lectura extendida.

Sobre la versión PDF para pantalla

La versión para pantalla se diferencia con relación a la de imprenta en que todos los hipervínculos quedan activos, permitiendo navegar el PDF en ambos sentidos.

Sobre la versión para lectura en línea

Esta versión está desarrollada con un deploy automático desde Git (en este caso GitHub Pages). Las familias tipográficas utilizadas dentro del libro son: IBM Plex, una superfamilia de tipografía abierta, diseñada y desarrollada conceptualmente por Mike Abbink en IBM con colaboración de Bold Monday y Georgia una tipografía con serifa diseñada en 1996 por Matthew Carter para Microsoft Corporation.
